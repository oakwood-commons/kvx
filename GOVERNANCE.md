# Governance

kvx is maintained by the project maintainer(s) listed in this file.

## Maintainers

- Adam Baker (GitHub: @abaker9)

## Decision Making

- The maintainer(s) review and merge pull requests.
- Releases are created by the maintainer(s).
- For contentious changes, the maintainer(s) will seek input in issues or discussions.

## Adding or Removing Maintainers

- Maintainers may add new maintainers by consensus.
- Maintainers may step down at any time by opening a pull request updating this file.
