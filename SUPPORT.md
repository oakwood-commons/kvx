# Support

Thanks for using kvx!

## Where to Get Help

- **Questions / How-to**: Open a GitHub Discussion
- **Bug Reports**: Open a GitHub Issue using the Bug Report template
- **Feature Requests**: Open a GitHub Issue using the Feature Request template

## Security Issues

Please follow the guidance in SECURITY.md and avoid disclosing vulnerabilities publicly.

## Response Expectations

Maintainers will respond as soon as possible. Response times may vary based on availability.
